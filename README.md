## Task
- Tác vụ 
	
	+ Tạo cụm đơn
	+ Tạo cụm HA

- Số Node
	+ Điều Kiện tối thiểu
		- Đối với chọn cụm đơn > 3 node tức >= 3
		- Đối với chọn cụm HA > 5 node

- Check
	+ Kiểm tra điều kiện
		- Thông số IP: > 2GRam, 2Vcpu

- Chọn NetWork

- Chọn kiểu Volume
